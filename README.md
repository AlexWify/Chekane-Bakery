# 🥐 Чеканэ CRM - Система управления пекарней

Веб-приложение для автоматизации работы пекарни: управление товарами, заказами, пользователями и ролями.

## 📋 Функционал

### 👤 Для всех пользователей
- **Регистрация и вход** по номеру телефона
- **Просмотр каталога** товаров с картинками
- **Корзина** с возможностью изменения количества
- **Оформление заказов**
- **Личный кабинет** с редактированием профиля
- **История заказов**

### 👑 Для администратора
- **Управление пользователями** (смена ролей)
- **Управление товарами** (добавление, редактирование, удаление)
- **Просмотр всех заказов** с фильтрацией по дате
- **Изменение статусов заказов**
- **Экспорт заказов в CSV**

### 💼 Для продавца
- **Просмотр всех заказов**
- **Изменение статусов заказов**
- **Экспорт заказов в CSV**

### 👨‍🍳 Для пекаря
- **Управление товарами** (добавление, редактирование, удаление)
- **Изменение статусов заказов**
